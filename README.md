# mstool_rshiny

## Prerequisited packages

-   library(Seurat)
-   library(tidyverse)
-   library(dplyr)
-   library(stringr)
-   library(ggplot2)
-   library(ggpmisc)
-   library(plotly)
-   library(ggpubr)
-   library(shiny)
-   library(DT)
-   library(shinydashboard)
-   library(shinyBS)
-   library(shinyalert)
-   library(markdown)
-   library(tools)
-   library(httr)
-   library(gprofiler2)
-   library(monocle3)

## How to

Simply open the "app.R" in Rstudio then click "Run App"

Or

In command line, do `runApp("app.R")`
